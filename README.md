## Description

A simple WORK IN PROGRESS app that will generate tabs based on some criteria, Powered by Nest.JS, the purpose of this project is also to learn how to create Nest.JS applications.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
